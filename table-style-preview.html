Table Style Preview

Five different visual directions for the class-program.html weekly grid, all rendered from the real Grade 7 — Faith schedule (pulled live from class-program.json). Scroll through and tell me which one — or which pieces of which ones — to carry into the real page.

01 · Broadsheet

Print timetable

Monochrome, hairline rules, serif display — reads like a page torn from a school prospectus. No color-coding; hierarchy comes purely from type weight and a single oxblood accent on the day names.

02 · Soft Cards

Friendly grid

No table borders at all — each period is its own rounded card in a CSS grid, pastel-tinted by subject family, with a soft shadow. Warmer and more approachable, closer to a parent-facing app than an admin sheet.

03 · Bold Blocks

Vectorized original

A cleaned-up, higher-contrast descendant of the scanned CLASS PROGRAM sheets this data came from — full-bleed saturated subject colors, a dark gutter, gradient day-headers. Loud, dense, and unmistakably a school timetable.

04 · Ops Dashboard

Live monitoring console

Dark, monospace, thin cyan grid lines — built around the fact this page already tracks the live "current period." The dashboard framing gives that feature somewhere to live: a pulsing "● LIVE" cell instead of a quiet outline.

05 · Day Agenda

Calendar timeline

Structurally different, not just restyled: five vertical day-columns like a calendar app's week view, with each period block's height proportional to its actual duration. Breaks read as thin gaps instead of full rows — the day's shape becomes visible at a glance.
